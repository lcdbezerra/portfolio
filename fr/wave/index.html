Wave Automação

Wave Automação a été un startup de développement de produits électroniques surtout dans la domotique. L'idée était développer des produits nationnels afin de les faire moins chers et devenir populaire ce marché au Brésil. Les produits étaient projetés d'une manière à n'avoir pas besoin d'installation d'une tierce personne, ce qui réduirait significativement les prix de ceux-ci en permettant aux utilisateurs d'installer eux-mêmes les produits, comme est déjà pratiqué aux États-Unis et en Europe.

Notre premier produit a été le Wave Plug, la prise intelligente. Sa fonction est surveiller la consommation de puissance électrique de quelque équipement lui connecté, et il permettre aussi que l'utilisateur allume ou éteind la prise à travers d'une application sur smartphone.

Le Wave Plug a communication WiFi en utilisant le module ESP8266-12E (programmé avec l'IDE Arduino), mesure le courant électrique avec le capteur ACS712, dont les données sont collectés par un microcontrôleur ATtiny85. Le module ESP8266 est le cerveau du système, puisqu'il accumule tous les paramètres.
